# Project Colors

## Core palette

| Usage | Color |
| --- | --- |
| Page background | `#eae7e3` |
| Primary heading / brand blue-gray | `#55657a` |
| Body text | `#4f5968` |
| Muted text | `#738296` |
| Card background | `#f5f2ee` |
| Warm border | `#d7d1ca` |
| Primary button | `#6683a8` |
| Primary button hover | `#203f68` |
| Bright/action blue | `#578cd6` |

## Notes

- Keep the site warm and muted rather than pure white/slate.
- Prefer `#eae7e3` for full-page backgrounds.
- Prefer `#f5f2ee` for cards and inset surfaces.
- Use `#55657a` for major headings and dark section backgrounds.
- Use `#6683a8` / `#203f68` for primary CTAs.
- Avoid generic Tailwind `slate-*` colors when building custom project sections unless a neutral fallback is needed.
